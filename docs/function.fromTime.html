Time

fromTime

Convert a Date Object to a "YYYY-MM-DDThh:mm:ss.sssZ" formatted string.

Syntax

CGU.fromTime(time)

time <Mixed>: Any time argument accepted by strftime/strfutc.

Return

<String>: A date string in the format "YYYY-MM-DDThh:mm:ss.sssZ".

(Other returns are the same as strftime/strfutc)
