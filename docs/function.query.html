Query

query

Search a query-string for all values of a key.

Syntax

CGU.query(key [, href])

key <String>: Name of the query-string variable.

href <String>: Any URI accepted by CGU.location.

<null>: Use the global location.

Return

<Array>: Query-string of an object.

<undefined>: Invalid href.

Examples

CGU.query('foo');  //=> <Array>
 
CGU.query('foo', 'foo=bar&red=foo');   //=> undefined (invalid href)
CGU.query('foo', '?foo=bar&red=foo');  //=> ['bar'] (complete href)
