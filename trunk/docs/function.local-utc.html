Time

local / utc

Absolute local and UTC time.

Syntax

CGU.local([yr [, mn [, dy [, hr [, mi [, sc [, ms]]]]]]])
CGU.utc([yr [, mn [, dy [, hr [, mi [, sc [, ms]]]]]]])

yr <Number>: Full Year (0 - 9999).

mn <Number>: Month (0 - 11).

dy <Number>: Date (1 - 31).

hr <Number>: Hour (0 - 23).

mi <Number>: Minute (0 - 59).

sc <Number>: Second (0 - 59).

ms <Number>: Millisecond (0 - 999).

<null>: Any unspecified values will default to that value of the current time.

Return

<Number>: Unix Epoch time in milliseconds.
