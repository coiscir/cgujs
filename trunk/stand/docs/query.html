Methods

read

Description: Get all values for a specified name (key).

Return: <Array>

Syntax: Query.read(key)

key <Mixed>: Name of the query-string variable. Any non-String keys will first be casted to String.

Examples: (Scenario: '?foo=bar&xyz=1&xyz=2')

Query.read('foo'); //=> ['bar']
Query.read('xyz'); //=> ['1', '2']
