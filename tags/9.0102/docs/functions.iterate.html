iterate

Iterate through each Object property or Array index.

Syntax

CGU.iterate(object, iterator [, instance])

object <Array>: Iterate through each Array index. object <Object>: Iterate through each Object property.

iterator <Function>: A function called for each property/index.

iterator(value, key, type)

instance <Boolean>: Limit iteration through instance properties. (optional)

Return

object: Returns the original object.

interrupt:

Interrupt

CGU.iterator expects undefined from iterator. Returning any other value from iterator will interrupt iteration.

var array = [9, 8, 7];

CGU.iterate(array, function (v) {
    alert(v);
});
//=> array

CGU.iterate(array, function (v) {
    return v;
});
//=> 9

Example

var o = {foo: 'bar', red: null};

var values = [];
CGU.iterate(o, function (v) {
  values.push(v);
});

var keys = [];
CGU.iterate(o, function (v, k) {
  keys.push(k);
});

// values => ['bar', null]
// keys   => ['foo', 'red']
