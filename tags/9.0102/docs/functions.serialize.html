serialize

Parameterize a Form’s elements.

Syntax

CGU.serialize(form [, complete])

form <Object>: Form DOM/Element Object or elements list.

complete <Boolean>: Return a complete location.search query-string. (optional)

Return

<String>: A query-string of the object.
