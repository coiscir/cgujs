Crypto

Crypto adds cryptographic hash functions to JavaScript.

It's designed to be simple to use, allowing 1-way and HMAC hashing through the use of a single function.

Functions

hash

Generate a cryptographic hash of given data.

hashes

Get a list of all available hash algorithms.

Encoding

Base-16, Base-32, and Base-64 are based on RFC 4648.

Base-16

base16

Uppercase hexadecimal (0-9 A-F).

hex

Lowercase hexadecimal (0-9 a-f).

Base-32

base32

Standard Base-32 (A-Z 2-7).

base32hex

Hexadecimal Base-32 (0-9 a-v).

Base-64

base64

Standard Base-64 (A-Z a-z 0-9 + /).

base64url

URL Alternative (A-Z a-z 0-9 - _).

Other

raw

Returns a byte array of the hash (e.g., [0x00, 0xff, ...]).

str

Returns an 8-bit encoded string via String.fromCharCode.
