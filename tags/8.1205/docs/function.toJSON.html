JSON

toJSON

Create a JSON-formatted string from an object.

Syntax

CGU.toJSON(input [, strict])

input <Mixed>: An object to be stringified.

strict <Boolean>: Only create JSON.org specified JSON strings.

Will not write undefined keyword.

Will not write object keys that are not string-formatted.

Return

<Mixed>: Actual type will depend on the JSON-string being parsed.

<SyntaxError>: json string is malformed. (thrown)

Example

CGU.toJSON({key: "value", foo: "bar", red: 24.42});        //=> {foo: "bar", key: "value", red: 24.42}
CGU.toJSON({key: "value", foo: "bar", red: 24.42}, true);  //=> {"foo": "bar", "key": "value", "red": 24.42}
 
// excludes Function and RegExp Objects
CGU.toJSON([function(){}, /pattern/, new Date(2008, 11, 31, 23, 59, 59)]);  //=> ["2009-01-01T05:59:59.000Z"]  // CST
